# azure-devops-automation-scripts
Azure DevOps Utilities: CLI Auth guide, bulk repository cloner, and CI/CD pipeline mass-updater scripts.
Este repositório contém ferramentas de automação que criei para resolver um problema de escala: precisávamos refatorar dezenas de pipelines CI/CD simultaneamente para injetar parâmetros de segurança (ex: Checkmarx/SonarQube). Usando esses scripts, reduzi um trabalho manual de horas para poucos minutos.
